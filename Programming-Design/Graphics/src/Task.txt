1. Реализовать графическое приложение "Игра 15"
    Есть квадратное графическое окно, которое занято костями (числами от 1 до 15) (по сути табличка 4 на 4, примыкающая к границам окна)
    Размер числа не менее 5/8 от высоты кости
    Кроме игрового поля есть меню

    При запуске приложения:
        1. Окно отображается в центре графического экрана
        2. Размеры окна изменить нельзя (менеджер компоновки сам должен компоновать кости)
        3. Игровое поле заполняется костями в случайном порядке, при этом нижнее правое место пустое

    Игрок, перемещая кости, должен собрать их в нужном порядке 1..4, 5..8, 9..12, 13..15

    Приложение должно давать возможность пользователю взаимодействовать с ним с помощью клавиатуры и мыши
    При щелчке кость перемещается на свободное место рядом с ней. (то есть вверх, вниз, влево, вправо)
    При использовании клавиатуры: кнопка вниз переместит верхнюю кость на пустое место и т.п. + должна быть возможность закрыть приложение с помощью клавиатуры


    МЕНЮ:
        1. Горизонтальное меню из пунктов File и Help
        2. Выпадающее меню File из пунктов New, Exit
        3. Выпадающее меню Help из пункта About
        Для всех пунктов меню задать мнемоники (просто буквы)
        Для New, Exit, About задать акселераторы (shortcuts)

        New - новое перемешивание
        Exit - закрывает окно и завершает приложение
        About - открывает модальное диалоговое окно в пределах окна приложение
                указать в нем ФИО, группу и текущий год

    Еще одно модальное окно отображается, когда пользователь собрал правильную комбинацию (оповещение)


используем toolkit чтобы узнать характеристики экрана
